## [1.0.0] - 2024-01-28

### Added
- Initial release of Sanjeevani Hospital Management System
- Multi-role authentication system (Admin, Doctor, Receptionist)
- Doctor management module (CRUD operations)
- Patient registration and management
- Receptionist management module
- Employee management module
- Appointment scheduling system
- OTP verification for secure operations
- SMS notification integration
- Password encryption for security
- Oracle database integration
- Java Swing GUI interface
- Splash screen and login system

### Features by Module

#### Admin Module
- Add, update, view, and remove doctors
- Add, update, view, and remove employees
- Add, update, view, and remove receptionists
- View all patients and appointments
- User management with role-based access control

#### Doctor Module
- View appointment requests
- Confirm or reject appointments
- View all confirmed appointments
- Access patient details and medical history
- Update appointment status

#### Receptionist Module
- Register new patients with detailed information
- Update existing patient records
- Schedule patient appointments
- Manage OPD assignments
- View patient lists

### Security Features
- Password hashing and encryption
- OTP-based verification
- Role-based access control (RBAC)
- Session management
- Secure database connections

### Technical Stack
- Java 17
- Oracle Database 11g/21c
- Java Swing for GUI
- Apache HttpClient for REST operations
- Unirest for API calls
- JSON processing library
